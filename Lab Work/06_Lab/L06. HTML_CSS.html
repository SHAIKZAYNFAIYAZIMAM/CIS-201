Questions

Q. Which is better, Documented CSS or External CSS? Why?

Ans: External CSS as it uses cleanly sperated CSS from HTML. They are stored Separately and are referenced in all document/pages. They can be stored anywhere. Can be imported from other web projects.

Q. Should we avoid inline CSS? Why?

Ans: Inline styles are not the best way to maintain your website. They go against every one of the best practices: Inline styles don't separate content from design: Inline styles are exactly the same as embedded font and other clunky design tags that modern developers rail against.
